Electronics

Geeky stuff here. A collection of hobbyist electronic projects I have done over the years. Many have been presented at the monthly meetings of the PIC Club, the Programming SIG which is part of the Sydney PC and Technology User Group where I'm a member.
  • Alarm Circuit for PC Parallel Port
    Interface an alarm circuit to the PC parallel printer port.

  • PICAXE Temperature Sensing with LM335
    Monitor temperature data in a GUI with Autoit.

  • Interrupting the PICAXE SERIN Command
    Use a slave PICAXE to interrupt SERIN command on master PICAXE.

  • How I built my quiet and cool PC
    Step-by-step guide to peace and quiet at home.

  • Alpha System Environment Monitor
    Temperature sensing on an Alpha box.

  • HAPSIM and the AVR Studio Simulator
    Fake LCD, LEDs, Push-Buttons and Keypad in AVR Simulator without hardware.

  • USB interface on an ATMEL ATtiny45
    Software implementation of the USB protocol on an ATtiny45.

  • .NET Micro Framework
    Programming Fez Panda II microcontroller with C# and Visual Studio.

  • Experimenting with DIY PCB
    My experiments in making a PCB using photoresist method.

  • Experimenting with DIY PCB - Toner Transfer
    More experiments in making a PCB using toner transfer method.

  • Waveshare 4.3" E-paper display
    Test-drive the demo code of this cool e-paper display.

  • DIY Arduino Tutorial
    Make your own Arduino and waste time and money doing it.

  • PAM8403 3W Stereo Amplifier
    Simple amplifier using a $1 PAM8403 chip from ebay.

  • LED Work Lamp and Switch Conversion
    Use cheap LED lights for work lamp. Convert latching switch to momentary

  • Blow the wall!
    A simple rocket shooter game using a 16x2 LCD, Arduino and a co-op task scheduler.

  • Laser Cutting
    Making a bamboo ply key chain with a laser cutter.

  • Amateur Radio
    Listening to Waverley Amateur Radio Club Monday night chat on UHF band.

  • Arduino Pro Mini with nRF24L01+ 2.4GHz radio
    Connect an nRF24L01+ transceiver to an Arduino Pro Mini and run some test sketches.

  • RS232 to TTL serial adapter
    Make an RS232 to TTL serial adapter similar to the Sparkfun FTDI Basic board but using a MAX233 chip.

  • Modify a Nerf Laser Ops Pro gun
    Turn a Nerf laser tag gun into a special effects weapon with awesome blaster shots and explosions.

  • Modify a Nerf Laser Ops Pro gun V2
    Add special sound effects to Nerf gun and play normal laser tag games with it as well.

  • Tooltips
    A collection of time and money-saving tools and tips for the electronics hobbyist.

  • DIY USBtinyISP-compatible AVR programmer
    An ATTINY85-based programmer for the Atmel AVR chip family similar to Adafruit's USBtinyISP.

  • Convert a 5V Arduino Nano clone to run on 3.3V
    No hardware board modifications required, just change the USB supply.

  • EBD-M05 Mini Electronic Load Tester
    The Zketech EBD-M05 is a mini electronic load for testing battery and powerbank discharge. A simple hack enables voltage and current to be measured during charging as well.

  • STM32 development boards revisited
    The STM32 dev boards have always been a bit of an underdog in the hobbyist world. Fast, cheap and easy-to-install: pick the first two only. Let's see if things have changed lately.

  • Simple USB sound system for PC
    A typical USB sound system for PC consists of 2 speakers with built-in amplifier and power adapter. Get rid of the adapter and free up a power outlet with this project.

  • LED Flip Switch powered from USB
    Modify the LED Flip Switch from Bunnings to use USB instead of battery power.

  • Control video game camera movement with thumb joystick
    Video games typically use the WASD keys to control an 'invisible' overhead camera which represents a player's overview of the battlefield. This project uses a joystick with an Arduino Pro Micro to emulate a HID keyboard instead.

  • Battery-powered Raspberry Pi with pan/tilt camera
    A portable Raspberry Pi 3B+ with a camera mount that can pan or tilt. Uses 6 x AA Nimh batteries and a high-efficiency DC-DC converter to step down to 5V.
