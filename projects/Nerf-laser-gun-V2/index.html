Modify a Nerf Laser Ops Pro Gun V2


This project adds new sound effects to a Nerf Laser Ops Pro gun by using a remote MP3 player module but preserves the original functionality of the gun. This means you can play normal laser tag games with it while having an enhanced audio experience. Check out the original project for a sound effects-only gun. The gun must be within range of the remote MP3 player since it relies on the 2.4Ghz nRF24L01+ transceiver for communications.


Laser Gun Mod V2

The gun has a 3.3V regulator so I dropped the original 3.3V LDO module and tapped into the onboard one instead. The laser pointer was also dropped since there is no space for it anymore. The gun's trigger and reload switch are connected to the Arduino in parallel by wires. When the trigger is pulled, the Arduino will detect a logic HIGH on its pin. The Nerf gun has a 10-shot limit before requiring the player to push the bottom switch for an ammo reload. The reload switch now has the additional function of being the weapon selector. If the switch is pressed and held, you can cycle through all available sound effects until you hear one you like. Upon release the selected sound will be the weapon default.

View the Arduino code in this link: Nerf Laser Gun V2





Remote MP3 Player V2

The remote MP3 player has been simplified with no second DFPlayer module to play the Star Trek theme song. An ESP32 has replaced the Arduino Nano which allowed the 3.3V regulator to be dropped. Connect the audio output jack to the line-in of an amplifier or hifi set using a stereo cable.

View the Arduino code in this link: Remote MP3 Player V2 ESP32




Space Nerf the final frontier

Press and hold the reload switch to go into weapons select mode and cycle through all available sound effects. Awesome! You can change the sounds by adding new MP3/WAV files to the 01 folder on the SD card. Then edit and upload the code in both gun and MP3 player accordingly.
