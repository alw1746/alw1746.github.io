Modify a Nerf Laser Ops Pro Gun


The Nerf Laser Ops Pro is a toy gun for kids that play laser tag games. Laser tag guns actually emit infrared light not laser so it is a bit of a marketing exercise. This one cost $10 secondhand but the sound effects are somewhat unimpressive due to the piezo speaker. Need to beef up the tinny blaster shots and add some feel good explosions.








Laser Gun Mod

So I hacked the gun by using an Arduino Pro Mini to send sound requests to a remote MP3 player with sound files on an SD card. An nRF24L01+ transceiver handles the wireless communications between the systems. The original switches are hooked up to the Arduino and a real laser pointer added for effects. Everytime the trigger is pulled, the current weapon's song number is sent to the remote MP3 player. The reload button triggers the Star Trek theme song as a background bonus. Note, this is now a special effects gun so you cannot play laser tag games with it.

View the Arduino code in this link: Nerf Laser Gun







Remote MP3 Player

The remote MP3 player receives a song number from the gun via the nRF24L01+ transceiver and issues the play command to the DFPlayer Mini MP3 Player module via a software serial port. There are 2 DFPlayer modules, one to play gun shot sounds, and the other the theme song. The audio from the 2 modules are mixed at the output stage. This is to work around the DFPlayer's limitation of playing only one sound file at a time. The 9V-to-5V stepdown is not really necessary, it works fine with a powerbank (add a 100R load resistor to prevent idle cutoff).
