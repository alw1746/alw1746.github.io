Arduino Pro Mini with nRF24L01+ 2.4GHz radio


The nRF24L01+ is a cheap ($5 AUD) 2.4GHz radio transceiver module widely available on ebay with a data transfer rate of 250kbps, 1Mbps and 2Mbps over a SPI interface. It runs on 3.3V but the i/o pins are 5V-tolerant which makes it easy to interface to an Arduino Pro Mini. There are plenty of online documentation and guides on how to use this transceiver. For example see this tutorial.



A 47uF bypass capacitor is used to smooth out power fluctuations that could affect its operation. Some recommend using a 100nf decoupling cap as well to filter out any high frequency noise in the power. All the cheap ebay modules are clones of the original Nordic Semiconductor chip but good enough for hobbyists to play with. Some are worse than others so do a range/reliability test to check them out.



The nRF24L01+ has a peak transmit current of 115mA so a stable power supply from a 500mA USB port is good. A 250mA 3.3V LDO regulator steps the 5V down for the transceiver.

My sketches below are customised from the original RF24 library which is well documented here.
  1. pingpair_ack.ino
    Range/reliability testing between 2 transceivers using a timed ping-pong (transmit-response) algorithm.

    2m apart in same room - output text file
    10m apart, separate rooms, 2 brick walls - output text file

  2. scanner.ino
    Scan and display 2.4GHz wifi channel traffic to help pick a 'quiet' channel.
    The default radio channel used by nRF24L01+ is 76(0x4C) which is on the edge of busy wifi traffic.
